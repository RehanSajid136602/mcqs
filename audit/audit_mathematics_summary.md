## Key Findings

### 1. Duplicate Correct Options (Chapter 1)
In **Chapter 1, Set 6, Question 6**, the options include both `2` and `$\sqrt[3]{8}$`. Both are mathematically identical and solve the inverse function query. While the marked index (1) is correct, having two identical correct answers should be avoided to prevent student confusion.

### 2. Ambiguous Definition of Linear Functions (Chapter 1)
In **Chapter 1, Set 5, Question 1**, the question suggests $f(x) = mx + c$ only represents a straight line when $m \neq 0$. In pure mathematics, $m = 0$ is a horizontal straight line. The explanation clarifies it refers to "varying values," but the question phrasing is slightly misleading.

### 3. Economic Applications (Chapter 3)
The formula for **Consumer Surplus** in **Chapter 3, Set 2, Question 12** uses a specific integral range $\int_p^b D(x) dx$. While valid under certain definitions of $b$, it differs from the more common $\int D(q)dq - pq$. This should be cross-referenced with the NBF Class 12 Mathematics textbook for consistency.
